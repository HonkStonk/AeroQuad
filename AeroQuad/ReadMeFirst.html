This sketch uses a modified version of the the ServoTimer2 library found at: 

http://www.arduino.cc/cgi-bin/yabb2/YaBB.pl?num=1204020386/75

 

To compile the AeroQuad sketch, the ServoTimer2 folder found in this distribution must be copied to ..\Arduino-00xx\hardware\libraries.

 

To verify the library has been successfully installed, execute Arduino xx and go to Sketch->Import Library.  You should see ServoTimer2 as one of the items to select (you don't need to select it at this time).

 

New features:

-          Single pole low pass filter implementation for reducing sensor noise

-          Improved calibration process for gyros and accelerometers

-          Additional user defined values able to be saved to EEPROM

-          Additional debug messages for troubleshooting

 

Happy flying!
